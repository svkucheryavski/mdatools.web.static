mdatools for R

R package for preprocessing, exploring and analysis of multivariate data. The package provides methods mostly common for Chemometrics. The package is in active developing with one-two major releases every year. Available both from CRAN and from GitHub.

  • Next major release: 0.11.0 (coming this summer)
  • Current stable version: 0.10.3
  • GitHub repository
  • User tutorial

Video lectures

Sets of videos which are used as part of real courses, where they are supplemented by quizzes, seminars, problem solving, etc. All videos available on YouTube, combined into playlists and sections. Some of the videos are outdated and need revision which is planned for autumn 2020.
