PCA app

The application allows to apply Principal Component Analysis to several datasets and play with main PCA results, including scores, loadings, residuals and explained variance interactively. The following datasets are currently available:

  • People dataset from Camo's book written by Kim Esbensen — 32 persons (16 males, 16 females) characterised by 12 parameters.
  • Wine dataset from PCA tutorial by R. Bro and A. Smilde — 44 wine samples characterised by 14 chemical parameters.
  • Octane dataset from Camo's book written by Kim Esbensen — NIR absorbance spectra (1100—1550 nm) of 39 gasoline samples with different octane number.

Main controls and gestures

2D plots

  • Drag and drop — select area and zoom in
  • Double click — reset zoom
